
## Installation

<details>
 
  <summary>install</summary>
  
```bash
npm install
```
  
</details>

<details>
 
  <summary>start mongodb server</summary>
  
```bash
sudo systemctl start mongod
```
  
</details>

<details>
 
  <summary>start server</summary>
  
```bash
npm start
```
  
</details>

URL: http://localhost:4000
